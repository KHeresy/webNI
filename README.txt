WebNI
=====

A WebSocket Server of OpenNI 2 and NiTE 2.

Library requirement:
1. Boost C++ Libraries
    http://www.boost.org/

2. WebSocket++
    http://www.zaphoyd.com/websocketpp

3. OpenNI 2
    http://www.openni.org/

3. PrimeSense NiTE 2
    http://www.openni.org/files/nite/

Build environment:
1. Windows 7 Pro
2. Microsoft Visual C++ 2012 Professional
